# 设计规范总结

本文档列出所有**固定的、统一的**设计标准，消除模糊和选择困难。

---

## 颜色规范

### 固定用法

| 用途 | 固定选择 | 说明 |
|------|---------|------|
| 操作按钮 | `bg-secondary text-secondary-foreground` | 橙色，所有主要操作统一用此 |
| 次要按钮 | `border border-input bg-background` | 白底描边 |
| 危险按钮 | `bg-destructive text-destructive-foreground` | 删除等危险操作 |
| 价格/重要数字 | `text-primary font-semibold` | 蓝色加粗 |
| 正文 | `text-foreground` | 深灰色 |
| 辅助说明 | `text-muted-foreground` | 中灰色 |
| 链接 | `text-primary hover:underline` | 蓝色，悬停下划线 |
| 卡片背景 | `bg-card` | 纯白（浅色模式） |
| 页面背景 | `bg-background` | 微蓝灰 |
| 边框 | `border-border` | 统一边框颜色 |
| 焦点环 | `focus:ring-2 focus:ring-ring` | 蓝色焦点环 |

## 字体规范

### 固定字体大小

| 元素类型 | 固定选择 | 说明 |
|---------|---------|------|
| 页面主标题 | `text-3xl font-bold` | 30px，每页一个 |
| 卡片标题 | `text-xl font-semibold` | 20px |
| 小节标题 | `text-lg font-medium` | 18px |
| 正文 | `text-base` | 16px，默认 |
| 按钮文字 | `text-base font-medium` | 16px |
| 标签/说明 | `text-sm` | 14px |
| 辅助信息 | `text-sm text-muted-foreground` | 14px，灰色 |
| 时间戳 | `text-xs text-muted-foreground` | 12px，灰色 |
| 价格（突出） | `text-2xl font-bold text-primary` | 24px，蓝色 |
| 价格（列表） | `text-xl font-semibold text-primary` | 20px，蓝色 |

### 响应式调整

```tsx
// 页面标题在移动端减小
<h1 className="text-2xl md:text-3xl font-bold">
  页面标题
</h1>

// 其他文字一般不需要响应式调整
```

## 间距规范

### 固定内边距（Padding）

| 元素类型 | 固定选择 | 像素值 |
|---------|---------|--------|
| 按钮 | `px-4 py-2` | 水平16px，垂直8px |
| 卡片 | `p-6` | 24px |
| 输入框 | `px-3 py-2` | 水平12px，垂直8px |
| 页面容器 | `px-4` | 左右16px |

### 固定元素间距

| 场景 | 固定选择 | 像素值 |
|------|---------|--------|
| 图标与文字 | `gap-2` | 8px |
| 按钮之间 | `gap-2` | 8px |
| 表单字段之间 | `space-y-4` | 16px |
| 表单分组之间 | `space-y-6` | 24px |
| 页面区块之间 | `space-y-6` | 24px |
| 卡片内容之间 | `space-y-4` | 16px |
| 列表项之间 | `space-y-2` | 8px |

### 固定外边距（Margin）

| 场景 | 固定选择 | 像素值 |
|------|---------|--------|
| 标题后 | `mb-4` | 下方16px |
| 段落后 | `mb-2` | 下方8px |
| 按钮区上方 | `mt-6` | 上方24px |

## 常见组件规范

### 标准按钮

```tsx

  立即预订
</button>

// 次要按钮（固定）
<button className="border border-input bg-background px-4 py-2 rounded-md text-base font-medium hover:bg-accent">
  取消
</button>

// 危险按钮（固定）
<button className="bg-destructive text-destructive-foreground px-4 py-2 rounded-md text-base font-medium hover:bg-destructive/90">
  删除

/>
```

## 常见问题

### Q: 什么时候可以偏离这些标准？

**A: 基本不偏离。** 这些是经过优化的标准值，适用于 95% 的场景。如果遇到特殊情况，请先咨询设计团队。

### Q: 为什么要统一标准？

**A: 三个原因：**
1. **一致性**：用户体验更流畅
2. **效率**：不用每次都思考选什么值
3. **维护**：修改时只需要改一处

### Q: 响应式设计怎么处理？

**A: 只在必要时使用响应式：**
- 页面标题：`text-2xl md:text-3xl`
- 页面边距：`px-4 md:px-6`
- 卡片内边距：`p-4 md:p-6`
- 其他元素：保持固定

### Q: 如果设计稿和标准不一致怎么办？

**A: 优先使用标准。** 如果设计稿确实有特殊需求，需要更新本标准文档，而不是直接实现非标准的设计。

## 检查清单

开发新组件或页面时，使用此清单确保符合规范：

- [ ] 按钮使用 `px-4 py-2 rounded-md`
- [ ] 操作按钮使用橙色 `bg-secondary`
- [ ] 卡片使用 `p-6 rounded-lg`
- [ ] 卡片标题使用 `text-xl font-semibold`
- [ ] 正文使用 `text-base`（或不写）
- [ ] 辅助说明使用 `text-sm text-muted-foreground`
- [ ] 价格使用 `text-2xl font-bold text-primary`
- [ ] 表单字段间距使用 `space-y-4`
- [ ] 输入框使用 `px-3 py-2 rounded-md border border-input`
- [ ] 输入框焦点使用 `focus:ring-2 focus:ring-ring`
- [ ] 页面容器使用 `container mx-auto px-4`
- [ ] 图标与文字使用 `gap-2`
