## 概述

定义应用的表单校验规则、错误提示、确认信息等系统消息,确保用户体验和文案风格的一致性。

## 全局消息

### 通用反馈提示

| 场景         | 类型      | 消息文本                               |
| :----------- | :-------- | :------------------------------------- |
| 通用网络错误 | `Error`   | `网络连接失败，请检查您的网络后重试。` |
| 通用服务错误 | `Error`   | `服务器开小差了，请稍后再试。`         |
| 通用成功操作 | `Success` | `操作成功`                             |

## 功能模块 UI 细节

### 用户注册

**表单字段校验:**

| 字段       | 校验规则             | 错误提示                       |
| :--------- | :------------------- | :----------------------------- |
| 手机号     | 必填                 | `请输入手机号`                 |
| 手机号     | 必须符合11位数字格式 | `手机号格式不正确，请重新输入` |
| 短信验证码 | 必填                 | `请输入短信验证码`             |
| 短信验证码 | 必须为6位数字        | `请输入正确的验证码`           |

**人机验证:**

| 场景                             | 提示文本         |
| :------------------------------- | :--------------- |
| 点击"发送验证码"后弹出的滑块验证 | `滑动以完成拼图` |

**协议相关:**

| 元素         | 文案内容                       |
| :----------- | :----------------------------- |
| 协议弹窗标题 | `携程用户注册协议和隐私政策`   |
| 同意按钮     | `同意并继续`                   |
| 不同意按钮   | `不同意`                       |

### 用户登录

**表单字段校验:**

| 字段 | 校验规则 | 错误提示           |
| :--- | :------- | :----------------- |
| 账户 | 必填     | `请输入用户名`     |
| 密码 | 必填     | `请输入登录密码`   |

**协议相关:**

| 场景           | 错误提示               |
| :------------- | :--------------------- |
| 未勾选用户协议 | `请先阅读并勾选协议`   |
