# 颜色系统

本项目使用蓝色+橙色的配色方案，采用 OKLCH 颜色空间。

**配色理念：**
- **蓝色（Primary）**: 专业、可信赖，用于主要内容和信息展示
- **橙色（Secondary）**: 活力、行动感，用于操作按钮和重要交互

---

## 快速参考

### 按钮颜色选择

| 按钮类型 | 背景色 | 文字颜色 | 说明 |
|---------|--------|---------|------|
| 主要操作（CTA） | `bg-secondary` | `text-secondary-foreground` | 橙色，吸引点击 |
| 次要操作 | `bg-background border border-input` | `text-foreground` | 白底描边 |
| 危险操作 | `bg-destructive` | `text-destructive-foreground` | 红色 |
| 幽灵按钮 | `bg-transparent hover:bg-accent` | `text-foreground` | 透明背景 |
| 链接按钮 | 无背景 | `text-primary hover:underline` | 蓝色文字 |

### 文字颜色选择

| 文字类型 | class | 说明 |
|---------|-------|------|
| 正文 | `text-foreground` | 深灰色 |
| 辅助说明 | `text-muted-foreground` | 中灰色 |
| 重要信息 | `text-primary` | 蓝色 |
| 价格/数字 | `text-primary font-bold` | 蓝色加粗 |
| 错误信息 | `text-destructive` | 红色 |
| 链接 | `text-primary hover:underline` | 蓝色 |

### 背景颜色选择

| 元素类型 | class | 说明 |
|---------|-------|------|
| 页面主背景 | `bg-background` | 微蓝灰 |
| 卡片 | `bg-card` | 纯白 |
| 操作按钮 | `bg-secondary` | 橙色 |
| 信息标签 | `bg-primary/10 text-primary` | 浅蓝底+蓝字 |
| 悬停效果 | `hover:bg-accent` | 浅蓝 |
| 禁用状态 | `bg-muted` | 浅灰 |

## 主题颜色

支持浅色和深色主题，自动适应用户偏好设置。

### Primary（主色 - 蓝色）

专业蓝色，用于主要内容、信息展示和导航。传达专业、可信赖的品牌形象。

**使用场景：**
- 品牌 Logo 和标识
- 重要信息展示（航班信息、价格标签）
- 导航链接和激活状态
- 信息类图标
- 状态指示（进行中、已确认）
- 超链接文字

**代码示例：**
```tsx
// 信息展示
<div className="text-primary font-semibold text-2xl">
  ¥2,580
</div>

// 激活的导航项
<NavLink className="text-primary font-medium border-b-2 border-primary">
  航班搜索
</NavLink>

// 信息图标
<InfoIcon className="text-primary" />
```

### Secondary（操作色 - 橙色）

活力橙色，用于所有操作按钮和需要用户点击的交互元素。吸引注意力，促进行动。

**使用场景：**

- 添加、创建等积极操作
- 需要强调的功能入口
- 悬浮操作按钮（FAB）

### Muted

弱化的背景色和辅助文字颜色，用于次要信息。

**使用场景：**
- 输入框占位符文字（`muted-foreground`）

- 次要信息标签
- 不活跃的导航项

// 禁用按钮

</Button>

## 使用指南

### 基本用法

使用 Tailwind 的语义化 class 名称：

```tsx
// 操作按钮 - 使用橙色
<button className="bg-secondary text-secondary-foreground hover:bg-secondary/90">
  立即预订
</button>

// 信息展示 - 使用蓝色
<div className="text-primary font-semibold text-xl">
  航班 CA1234
</div>

// 次要按钮 - 使用边框样式
<button className="border border-input bg-background hover:bg-accent hover:text-accent-foreground">
  查看详情
</button>

// 危险操作 - 使用红色
<button className="bg-destructive text-destructive-foreground hover:bg-destructive/90">
  取消订单
</button>

// 卡片容器
<div className="bg-card text-card-foreground border border-border rounded-lg p-4">
  <h3 className="font-semibold">标题</h3>
  <p className="text-muted-foreground">辅助说明</p>
</div>

// 输入框
<input className="bg-background text-foreground border border-input focus:ring-2 focus:ring-ring rounded px-3 py-2" />
```

### 透明度修饰符

调整颜色的透明度，数字表示不透明度百分比：

```tsx
<div className="bg-primary/10">10% 不透明度</div>
<div className="bg-primary/50">50% 不透明度</div>
<div className="bg-primary/90">90% 不透明度</div>

// 常用于悬停效果
<button className="bg-primary hover:bg-primary/90">按钮</button>
```
