# 颜色系统

本项目使用基于 Tailwind CSS 的语义化颜色系统，采用 OKLCH 颜色空间。

## 主题颜色

支持浅色和深色主题，自动适应用户偏好设置。

---

### Primary（主色）

品牌主色，用于最重要的操作和元素。

**使用场景：**
- 主要操作按钮（预订、提交、确认、购买）
- 关键导航链接
- 激活状态（选中的标签页、当前页面导航）
- 进度条、加载指示器
- 需要强调的图标和徽章

**代码示例：**
```tsx
// 主要按钮
<Button className="bg-primary text-primary-foreground">
  预订航班
</Button>

// 激活的导航项
<NavLink className="text-primary font-medium">
  我的订单
</NavLink>

// 进度条
<div className="h-2 bg-primary rounded-full" style={{width: '60%'}} />
```

### Muted

弱化的背景色和辅助文字颜色，用于次要信息。

**使用场景：**
- 输入框占位符文字（`muted-foreground`）
- 禁用状态的元素
- 辅助说明文字、时间戳
- 次要信息标签
- 不活跃的导航项
- 微妙的背景区域（`muted`）

**代码示例：**
```tsx
// 表单字段说明
<div>
  <Label>邮箱地址</Label>
  <Input placeholder="请输入邮箱" />
  <p className="text-sm text-muted-foreground mt-1">
    我们会向该邮箱发送确认信息
  </p>
</div>

// 时间戳
<p className="text-xs text-muted-foreground">2 小时前</p>

// 禁用按钮
<Button disabled className="bg-muted text-muted-foreground">
  不可用
</Button>
```

用于数据可视化、图表和统计数据展示。使用 `chart-1` 到 `chart-5` 来区分不同的数据系列。

## 侧边栏颜色

侧边栏专用的颜色系统，与主内容区域有所区分。

**使用场景：**
- `sidebar`: 侧边栏背景
- `sidebar-foreground`: 侧边栏普通文字
- `sidebar-primary`: 侧边栏中的主要元素（当前页面）
- `sidebar-accent`: 侧边栏悬停状态
- `sidebar-border`: 侧边栏边框和分割线

**代码示例：**
```tsx
// 侧边栏容器
<aside className="bg-sidebar text-sidebar-foreground border-r border-sidebar-border">
  {/* 当前激活的导航项 */}
  <a className="bg-sidebar-primary text-sidebar-primary-foreground">
    首页
  </a>
  
  {/* 普通导航项 */}
  <a className="hover:bg-sidebar-accent hover:text-sidebar-accent-foreground">
    订单管理
  </a>
</aside>
```

## 使用指南

### 基本用法

使用 Tailwind 的语义化 class 名称：

```tsx
// 主要按钮
<button className="bg-primary text-primary-foreground hover:bg-primary/90">
  点击我
</button>

// 次要按钮
<button className="bg-secondary text-secondary-foreground hover:bg-secondary/80">
  取消
</button>

// 破坏性操作
<button className="bg-destructive text-destructive-foreground hover:bg-destructive/90">
  删除
</button>

// 卡片容器
<div className="bg-card text-card-foreground border border-border rounded-lg p-4">
  <h3 className="font-semibold">卡片标题</h3>
  <p className="text-muted-foreground">这是一段辅助说明文字</p>
</div>

// 输入框
<input className="bg-background text-foreground border border-input focus:ring-2 focus:ring-ring" />
```

### 透明度修饰符

调整颜色的透明度，数字表示不透明度百分比：

```tsx
<div className="bg-primary/10">10% 不透明度</div>
<div className="bg-primary/50">50% 不透明度</div>
<div className="bg-primary/90">90% 不透明度</div>

// 常用于悬停效果
<button className="bg-primary hover:bg-primary/90">按钮</button>
```

## 快速参考

### 按钮颜色选择

| 按钮类型 | 背景色 | 文字颜色 |
|---------|--------|---------|
| 主要操作 | `bg-primary` | `text-primary-foreground` |
| 次要操作 | `bg-secondary` | `text-secondary-foreground` |
| 危险操作 | `bg-destructive` | `text-destructive-foreground` |
| 幽灵按钮 | `bg-transparent` | `text-foreground` |
| 链接按钮 | 无背景 | `text-primary` |

### 文字颜色选择

| 文字类型 | class |
|---------|-------|
| 正文 | `text-foreground` |
| 辅助说明 | `text-muted-foreground` |
| 重要信息 | `text-primary` |
| 错误信息 | `text-destructive` |
| 链接 | `text-primary hover:underline` |

### 背景颜色选择

| 元素类型 | class |
|---------|-------|
| 页面主背景 | `bg-background` |
| 卡片 | `bg-card` |
| 次要区域 | `bg-secondary` |
| 强调区域 | `bg-accent` |
| 悬停效果 | `hover:bg-accent` |
