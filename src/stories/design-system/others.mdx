本文档涵盖间距系统、圆角规范、阴影效果和布局原则。

## 间距系统

使用 Tailwind 的 4px 基础间距系统，所有间距都是 4 的倍数。

### 间距单位对照表

| Class | 值 | 常用场景 |
|-------|-----|----------|
| `0.5` | 2px | 极小间距（很少用） |
| `1` | 4px | 图标与文字的紧密间距 |
| `2` | 8px | 按钮内文字与边缘、小型元素间距 |
| `3` | 12px | 中等间距 |
| `4` | 16px | 标准元素间距、表单字段间距 |
| `5` | 20px | 较大间距 |
| `6` | 24px | 区块间距、卡片内边距 |
| `8` | 32px | 大区块间距、页面分区 |
| `10` | 40px | 超大间距 |
| `12` | 48px | 页面顶部/底部间距 |

### 内边距（Padding）

用于元素内部的空间。

```tsx
// 按钮内边距 - 统一标准
<button className="px-4 py-2">标准按钮</button>

// 卡片内边距 - 统一 p-6
<div className="p-6">标准卡片（24px）</div>

// 不同方向的内边距
<div className="pt-4 pb-6">上4下6</div>
<div className="px-4 py-2">左右4上下2</div>
```

### 外边距（Margin）

用于元素之间的间距。

```tsx
// 单个元素的外边距
<h2 className="mb-4">标题（下方16px间距）</h2>
<p className="mt-2">段落（上方8px间距）</p>

// 负边距（特殊情况使用）
<div className="-mt-4">向上偏移16px</div>
```

### 间距组合（Space / Gap）

更推荐使用 `space-*` 或 `gap-*` 来处理子元素间距。

```tsx
// 垂直间距 - 使用 space-y
<div className="space-y-4">
  <div>元素1</div>
  <div>元素2</div>
  <div>元素3</div>
</div>

// 水平间距 - 使用 space-x
<div className="flex space-x-2">
  <button>按钮1</button>
  <button>按钮2</button>
</div>

// Flexbox/Grid 间距 - 使用 gap
<div className="flex gap-4">
  <div>项目1</div>
  <div>项目2</div>
</div>

<div className="grid grid-cols-3 gap-6">
  <div>卡片1</div>
  <div>卡片2</div>
  <div>卡片3</div>
</div>
```

### 常见布局模式

```tsx
// 表单布局
<form className="space-y-6">
  <div className="space-y-2">
    <label className="block">姓名</label>
    <input />
    <p className="text-sm text-muted-foreground">辅助文字</p>
  </div>
  
  <div className="space-y-2">
    <label className="block">邮箱</label>
    <input />
  </div>
  
  <div className="flex gap-2 justify-end">
    <button>取消</button>
    <button>提交</button>
  </div>
</form>

// 卡片内部布局
<div className="bg-card border border-border rounded-lg p-6 space-y-4">
  <h3 className="text-xl font-semibold">标题</h3>
  <p className="text-muted-foreground">描述内容</p>
  <div className="flex gap-2">
    <button>操作1</button>
    <button>操作2</button>
  </div>
</div>

// 页面布局
<div className="min-h-screen bg-background">
  {/* 页面内边距 */}
  <div className="container mx-auto px-4 py-8">
    {/* 区块间距 */}
    <div className="space-y-8">
      <section>第一部分</section>
      <section>第二部分</section>
      <section>第三部分</section>
    </div>
  </div>
</div>
```

### 响应式间距

```tsx
// 移动端小，桌面端大
<div className="p-4 md:p-6 lg:p-8">
  响应式内边距
</div>

// 移动端紧凑，桌面端宽松
<div className="space-y-4 md:space-y-6">
  <div>内容1</div>
  <div>内容2</div>
</div>
```

## 圆角系统

基础圆角值：`--radius: 0.75rem` (12px)

### 圆角大小对照表

| CSS 变量 | Tailwind Class | 计算值 | 使用场景 |
|----------|----------------|--------|----------|
| `--radius-sm` | `rounded-sm` | 8px | 小型元素、徽章、标签 |
| `--radius-md` | `rounded-md` | 10px | 按钮、输入框 |
| `--radius-lg` | `rounded-lg` | 12px | 卡片、对话框 |
| `--radius-xl` | `rounded-xl` | 16px | 大型容器、图片 |
| - | `rounded-full` | 完全圆角 | 头像、圆形按钮、药丸状标签 |
| - | `rounded-none` | 0 | 特殊情况（通常不用） |

### 使用示例

```tsx
// 按钮 - rounded-md (10px)
<button className="bg-secondary text-secondary-foreground px-4 py-2 rounded-md">
  按钮
</button>

// 输入框 - rounded-md (10px)
<input className="border border-input px-3 py-2 rounded-md" />

// 卡片 - rounded-lg (12px)
<div className="bg-card border border-border rounded-lg p-6">
  卡片内容
</div>

// 航班卡片 - rounded-lg
<div className="bg-card border border-border rounded-lg p-6 space-y-4">
  <div className="text-lg font-medium">CA1234</div>
  <div className="text-2xl font-semibold">08:00 - 10:30</div>
</div>

// 徽章/标签 - rounded-full（完全圆角）
<span className="inline-flex items-center px-3 py-1 rounded-full bg-primary/10 text-primary text-sm font-medium">
  已确认
</span>

// 头像 - rounded-full
<img 
  src="/avatar.jpg" 
  className="w-10 h-10 rounded-full" 
  alt="头像" 
/>

// 图片卡片 - rounded-xl
<div className="overflow-hidden rounded-xl">
  <img src="/flight.jpg" alt="航班" />
</div>

// 对话框 - rounded-lg
<div className="bg-background border border-border rounded-lg shadow-lg p-6 max-w-md">
  <h3 className="text-xl font-semibold mb-4">确认订单</h3>
  <p className="text-muted-foreground mb-6">确定要预订此航班吗？</p>
  <div className="flex gap-2 justify-end">
    <button className="px-4 py-2 rounded-md border border-input">取消</button>
    <button className="px-4 py-2 rounded-md bg-secondary text-secondary-foreground">确认</button>
  </div>
</div>
```

### 单边圆角

某些情况下只需要部分圆角：

```tsx
// 只有顶部圆角
<div className="rounded-t-lg">顶部圆角</div>

// 只有底部圆角
<div className="rounded-b-lg">底部圆角</div>

// 只有左侧圆角
<div className="rounded-l-lg">左侧圆角</div>

// 只有右侧圆角
<div className="rounded-r-lg">右侧圆角</div>

// 单个角
<div className="rounded-tl-lg">左上角</div>
<div className="rounded-tr-lg">右上角</div>
<div className="rounded-bl-lg">左下角</div>
<div className="rounded-br-lg">右下角</div>
```

### 实际应用场景

```tsx
// 带图片的卡片
<div className="bg-card border border-border rounded-lg overflow-hidden">
  <img src="/cover.jpg" className="w-full h-48 object-cover" />
  <div className="p-6">
    <h3 className="text-xl font-semibold">标题</h3>
    <p className="text-muted-foreground mt-2">描述</p>
  </div>
</div>

// 按钮组
<div className="inline-flex rounded-lg border border-input overflow-hidden">
  <button className="px-4 py-2 bg-accent">选项1</button>
  <button className="px-4 py-2 border-l border-input">选项2</button>
  <button className="px-4 py-2 border-l border-input">选项3</button>
</div>
```

## 阴影系统

使用 Tailwind 默认阴影系统，增强元素层次感。

### 阴影级别

| Class | 效果 | 使用场景 |
|-------|------|----------|
| 无 | 无阴影 | 使用 `border` 的平面卡片 |
| `shadow-sm` | 微妙阴影 | 微微悬浮的卡片 |
| `shadow` | 标准阴影 | 一般卡片、悬浮元素 |
| `shadow-md` | 中等阴影 | 下拉菜单、弹出层 |
| `shadow-lg` | 深阴影 | 对话框、模态框 |
| `shadow-xl` | 超深阴影 | 全屏对话框 |
| `shadow-2xl` | 最深阴影 | 特殊强调元素 |

### 使用示例

### 使用示例

```tsx
// 平面卡片（使用边框，无阴影）
<div className="bg-card border border-border rounded-lg p-6">
  平面设计，使用边框区分
</div>

// 微妙悬浮卡片
<div className="bg-card rounded-lg shadow-sm p-6">
  轻微悬浮效果
</div>

// 标准卡片
<div className="bg-card rounded-lg shadow p-6">
  标准悬浮卡片
</div>

// 下拉菜单
<div className="bg-popover rounded-lg shadow-md p-2">
  <button className="w-full px-4 py-2 text-left hover:bg-accent rounded-md">选项1</button>
  <button className="w-full px-4 py-2 text-left hover:bg-accent rounded-md">选项2</button>
</div>

// 对话框
<div className="bg-background rounded-lg shadow-lg p-6 max-w-md">
  <h3 className="text-xl font-semibold">对话框标题</h3>
  <p className="text-muted-foreground mt-2">对话框内容</p>
  <div className="flex gap-2 justify-end mt-6">
    <button className="px-4 py-2 rounded-md border border-input">取消</button>
    <button className="px-4 py-2 rounded-md bg-secondary text-secondary-foreground">确认</button>
  </div>
</div>
```

### 悬停效果

```tsx
// 卡片悬停时增加阴影
<div className="bg-card border border-border rounded-lg p-6 hover:shadow-md transition-shadow cursor-pointer">
  可交互卡片
</div>

// 按钮悬停时增加阴影
<button className="bg-secondary text-secondary-foreground px-4 py-2 rounded-md hover:shadow transition-shadow">
  按钮
</button>
```

### 深色模式下的阴影

深色模式下阴影效果不明显，建议配合边框使用：

```tsx
// 在深色模式下也清晰的卡片
<div className="bg-card border border-border rounded-lg p-6 shadow-sm">
  内容
</div>
```

## 布局原则

### 容器宽度

```tsx
// 全宽容器
<div className="w-full">全宽</div>

// 响应式容器（Tailwind container）
<div className="container mx-auto px-4">
  自动居中，带左右内边距
</div>

// 最大宽度限制
<div className="max-w-7xl mx-auto px-4">
  最大宽度 1280px
</div>

<div className="max-w-4xl mx-auto px-4">
  内容区域，最大宽度 896px
</div>

<div className="max-w-md mx-auto px-4">
  表单/对话框，最大宽度 448px
</div>
```

### 栅格布局

```tsx
// 简单两列布局
<div className="grid grid-cols-2 gap-4">
  <div>列1</div>
  <div>列2</div>
</div>

// 响应式栅格（移动端1列，平板2列，桌面3列）
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <div>卡片1</div>
  <div>卡片2</div>
  <div>卡片3</div>
</div>

// 航班搜索结果列表（卡片式布局）
<div className="grid grid-cols-1 gap-4">
  <div className="bg-card border border-border rounded-lg p-6">航班1</div>
  <div className="bg-card border border-border rounded-lg p-6">航班2</div>
  <div className="bg-card border border-border rounded-lg p-6">航班3</div>
</div>
```

### Flexbox 布局

```tsx
// 水平排列
<div className="flex items-center gap-4">
  <div>项目1</div>
  <div>项目2</div>
</div>

// 两端对齐
<div className="flex items-center justify-between">
  <div>左侧内容</div>
  <div>右侧内容</div>
</div>

// 垂直居中
<div className="flex items-center justify-center min-h-screen">
  <div>居中内容</div>
</div>

// 卡片内部布局（常见）
<div className="bg-card border border-border rounded-lg p-6">
  <div className="flex items-start justify-between mb-4">
    <div>
      <h3 className="text-xl font-semibold">标题</h3>
      <p className="text-sm text-muted-foreground">副标题</p>
    </div>
    <div className="text-2xl font-bold text-primary">¥580</div>
  </div>
  <div className="flex gap-2">
    <button className="flex-1 border border-input rounded-md py-2">详情</button>
    <button className="flex-1 bg-secondary text-secondary-foreground rounded-md py-2">预订</button>
  </div>
</div>
```

### 页面布局模式

```tsx
// 基础页面布局
<div className="min-h-screen bg-background">
  {/* 顶部导航 */}
  <header className="border-b border-border bg-card">
    <div className="container mx-auto px-4 py-4">
      导航内容
    </div>
  </header>
  
  {/* 主内容区 */}
  <main className="container mx-auto px-4 py-8">
    <div className="max-w-4xl mx-auto space-y-8">
      <section>内容区域</section>
    </div>
  </main>
  
  {/* 底部 */}
  <footer className="border-t border-border bg-card mt-auto">
    <div className="container mx-auto px-4 py-6">
      底部内容
    </div>
  </footer>
</div>

// 侧边栏布局
<div className="min-h-screen bg-background">
  <div className="flex">
    {/* 侧边栏 */}
    <aside className="w-64 border-r border-border bg-sidebar">
      侧边栏内容
    </aside>
    
    {/* 主内容 */}
    <main className="flex-1 p-8">
      主要内容
    </main>
  </div>
</div>
```

## 断点系统

Tailwind 默认断点：

| 断点 | 最小宽度 | 设备类型 |
|------|----------|----------|
| `sm` | 640px | 大型手机 |
| `md` | 768px | 平板 |
| `lg` | 1024px | 小型笔记本 |
| `xl` | 1280px | 桌面 |
| `2xl` | 1536px | 大屏幕 |

### 响应式使用示例

```tsx
// 响应式栅格
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  卡片
</div>

// 响应式间距
<div className="p-4 md:p-6 lg:p-8">
  内容
</div>

// 响应式文字
<h1 className="text-2xl md:text-3xl lg:text-4xl">
  标题
</h1>

// 响应式显示/隐藏
<div className="hidden md:block">
  桌面端显示
</div>

<div className="block md:hidden">
  移动端显示
</div>
```
