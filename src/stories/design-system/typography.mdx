# 字体系统

## 字体族

本项目使用 Geist 字体系列，现代、清晰、易读。

| 用途 | CSS 变量 | Tailwind Class | 说明 |
|------|----------|----------------|------|
| 正文、UI | `--font-geist-sans` | `font-sans` | 默认字体，无需显式声明 |
| 代码、数字 | `--font-geist-mono` | `font-mono` | 等宽字体 |

---

## 快速参考

### 我应该用什么字体大小？

**这是标题吗？**
- 页面主标题 → `text-3xl font-bold`
- 卡片/分组标题 → `text-xl font-semibold`
- 小节标题 → `text-lg font-medium`

**这是正文吗？**
- 主要内容 → `text-base font-normal`（默认，不用写）
- 强调内容 → `text-base font-medium`
- 次要说明 → `text-sm text-muted-foreground`

**这是按钮吗？**
- 标准按钮 → `text-base font-medium`
- 小按钮 → `text-sm font-medium`

**这是价格/数字吗？**
- 突出显示 → `text-2xl font-bold text-primary`
- 列表中 → `text-xl font-semibold text-primary`

**这是辅助信息吗？**
- 说明文字 → `text-sm text-muted-foreground`
- 时间戳 → `text-xs text-muted-foreground`

## 字体大小

### 标题层级

| Class | 大小 | 行高 | 使用场景 | 示例 |
|-------|------|------|----------|------|
| `text-4xl` | 36px | 40px | 页面主标题 | 航班搜索结果页 |
| `text-3xl` | 30px | 36px | 一级标题 | 预订确认、重要页面标题 |
| `text-2xl` | 24px | 32px | 二级标题 | 卡片标题、表单分组标题 |
| `text-xl` | 20px | 28px | 三级标题 | 列表项标题、小节标题 |

**使用建议：**
- 每个页面只有一个 `text-3xl` 标题
- 标题使用 `font-semibold` 字重
- 移动端可适当减小：`text-2xl md:text-3xl`

---

### 正文层级

| Class | 大小 | 行高 | 使用场景 | 示例 |
|-------|------|------|----------|------|
| `text-lg` | 18px | 28px | 强调正文、引导文案 | 表单说明、重要提示 |
| `text-base` | 16px | 24px | 标准正文 | 默认文字大小，大多数内容 |
| `text-sm` | 14px | 20px | 次要信息、标签 | 表单标签、列表说明 |
| `text-xs` | 12px | 16px | 辅助信息、提示 | 时间戳、版权信息、徽章 |

**使用建议：**
- `text-base` 是默认大小，无需显式声明
- 同一组件内最多使用 3 种字体大小
- 正文使用 `font-normal` 字重

---

## 字重

| Class | 字重值 | 使用场景 | 示例 |
|-------|--------|----------|------|
| `font-normal` | 400 | 正文、描述性内容 | 段落文字、列表说明 |
| `font-medium` | 500 | 按钮文字、表单标签 | 所有按钮、Input label |
| `font-semibold` | 600 | 小标题、重要信息 | 卡片标题、价格 |
| `font-bold` | 700 | 主标题、强调数字 | 页面标题、金额显示 |

**使用建议：**
- 按钮统一使用 `font-medium`
- 价格/金额使用 `font-semibold`
- 避免在正文中使用 `font-bold`（显得突兀）

## 常见场景示例

```tsx

<h1 className="text-4xl font-bold text-foreground">
  搜索航班
</h1>

  <h1 className="text-3xl font-bold text-foreground">我的订单</h1>

    查看和管理您的所有订单
  </p>

<h1 className="text-2xl md:text-4xl font-bold text-foreground">

</h1>
```

## 文本颜色搭配

### 正文颜色

| 颜色 | Class | 使用场景 |
|------|-------|----------|
| 主要文字 | `text-foreground` | 所有正常文字内容 |
| 辅助文字 | `text-muted-foreground` | 说明、注释、时间戳 |

### 强调颜色

| 颜色 | Class | 使用场景 |
|------|-------|----------|
| 蓝色强调 | `text-primary` | 价格、航班号、重要信息 |
| 链接 | `text-primary hover:underline` | 可点击的文字链接 |
| 橙色强调 | `text-secondary` | 提醒、待处理状态 |
| 错误 | `text-destructive` | 错误提示、验证失败 |
| 成功 | `text-chart-5` | 成功提示、已完成 |

## 响应式字体

### 移动端优化

```tsx
{/* 标题在移动端减小 */}
<h1 className="text-2xl md:text-4xl font-bold">
  页面标题
</h1>

{/* 正文在移动端保持不变 */}
<p className="text-base">
  正文内容移动端和桌面端大小一致
</p>

{/* 小字在移动端保持清晰 */}
<span className="text-sm md:text-base">
  次要信息
</span>
```

### 断点建议

- `text-xs` / `text-sm`: 移动端和桌面端保持一致
- `text-base`: 所有尺寸保持一致（标准大小）
- `text-lg` 及以上: 移动端可适当减小 1-2 级

---

## 行高和间距

### 行高原则

- 标题：较紧凑的行高（1.2-1.3 倍）
- 正文：舒适的行高（1.5-1.6 倍）
- 列表/卡片：标准行高（Tailwind 默认）

### 段落间距

```tsx
{/* 标题和内容之间 */}
<h2 className="text-xl font-semibold">标题</h2>
<p className="text-base mt-2">内容紧跟标题用 mt-2</p>

{/* 段落之间 */}
<div className="space-y-4">
  <p>第一段</p>
  <p>第二段</p>
</div>

{/* 表单字段之间 */}
<div className="space-y-6">
  <div>字段1</div>
  <div>字段2</div>
</div>
```
